这不是什么现成的软件，是我用 Claude Code 从零开发的桌面宠物，卡比兽人格！它能通过 Hooks 实时感知你的编程状态，还会自言自语、提醒你喝水吃饭，甚至直接跟 Claude Code 聊天。接下来我带你看看，这个东西是怎么做出来的。

## 第2段：灵感与架构（0:45-2:00）

做这个项目的出发点其实很简单——一个人写代码的时候，尤其是用 AI 编程工具的时候，屏幕上只有冷冰冰的终端输出。你不知道 AI 在干嘛，它也不知道你在干嘛。这种感觉很孤独。

所以我就想，能不能让 AI 编程的过程变得「看得见」？不是看日志，而是有一个小家伙，它就在你桌面上，跟你一起经历整个开发过程。这个家伙得有点性格——于是选了卡比兽。贪吃、爱睡、但又关心主人健康，完美。

整个架构其实特别简洁，就一条线——

Claude Code 的 Hooks 机制在关键节点触发回调——你提交指令、工具调用开始、工具调用结束、任务完成。Hook 脚本把这些事件转成 HTTP 请求，发到本地 Flask 服务器，再通过队列推给 Tkinter 界面驱动状态切换和动画。

但这只是基础。我们还加了三个独立模块：闹钟管理器在后台每 30 秒检查系统时间；自言自语循环每 1 到 3 分钟随机冒泡卖萌；交谈模块则直接调用 Claude CLI 实现双向对话。

---

## 第3段：核心实现（2:00-4:30）

### 3.1 Hooks 配置与人格系统（2:00-2:50）

先看入口——Hooks 配置。Claude Code 支持四种 Hook 事件，每个事件都指向同一个 Python 脚本。

配置很简单，指定事件类型和要执行的命令就行。timeout 设 5 秒，不能阻塞主流程。

然后是人格系统。卡比兽有 9 种状态，每种状态的标签都是卡比兽风格："呼噜噜~"、"咔比正在想..."、"困困...想睡觉( '・ω・)゛"。单击它会根据状态说不同的话，全是颜文字风格。

### 3.2 事件映射与自言自语（2:50-3:40）

这个脚本做的事情很纯粹——从 stdin 读取 Hook 传来的 JSON，判断事件类型，然后发 HTTP POST 到本地宠物应用。

注意这个过滤逻辑——不是所有工具调用都值得通知。Read、Grep 这种查询操作不需要反应，但 Write、Edit、Bash 才让宠物切换到"阶段完成"。

还有一个有趣的功能——自言自语。宠物有 52 条卡比兽风格的随机短语，涵盖吃东西、喝水提醒、犯困、陪主人、天气、鼓励、卖萌、下午茶。每 1 到 3 分钟随机弹一条，但睡觉时不会弹。

### 3.3 状态机与动画引擎（3:40-4:30）

宠物有 9 种状态，每种对应不同的颜色、文字和动画。

动画引擎核心是 50 毫秒一次的循环，根据状态用不同数学函数计算偏移量。摸鱼是正弦浮动，工作中是双频叠加颤抖，完成时是弹跳。整个动画系统就 30 多行代码，但效果丰富——程序化动画的威力。

## 第4段：功能展示（4:30-6:00）

来看完整的效果。

### 4.1 基础交互（4:30-5:00）

单击卡比兽，它会根据状态说不同的话。双击——弹出搜狗输入法风格的迷你浮条，输入任何问题按 Enter，卡比兽在后台调用 Claude CLI，回复以卡通气泡显示在头顶。气泡是奶油色圆角矩形、暖棕边框、还有 45 度倾斜的猫爪装饰。

### 4.2 闹钟系统（5:00-5:30）

右键菜单里有个"闹钟设置"。10 个预设模板：上午 10 点喝水、11 点补充水分、12 点吃午饭、下午 2 点喝水、3 点休息活动、4 点喝水、5 点半准备下班、6 点下班、7 点吃晚饭。到点弹出提醒窗口，有系统提示音，还可以"5 分钟后提醒"贪睡。

### 4.3 智能睡眠（5:30-6:00）

卡比兽有个智能机制——如果 10 分钟没有点它或者没有任何 Claude Code 操作，它就自动进入睡眠状态。但一旦你回来操作，它立刻醒来陪你。

---

## 第5段：效率复盘（6:00-7:00）

最后聊聊开发效率。这个项目包含 Flask HTTP 服务、Tkinter GUI、9 种状态的状态机、9 种动画、Hook 脚本、闹钟系统、对话模块、自言自语、卡通气泡绘制——功能不少。

如果纯手写，光状态机和动画就要折腾一整天。但用 Claude Code，核心功能从想法到跑起来大概两三个小时。它帮我搞定了架构设计、状态机实现、动画算法这些模式化部分，我主要精力花在调动画参数和人格设定上。

不过也得说实话——AI 生成的代码不是拿来就能用的。动画的振幅和速度需要手动微调，卡比兽的每句台词也是我一条条写的。AI 是加速器不是替代品，但确实把重复性工作压缩了很多。

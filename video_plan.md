## 一、视频主题与核心卖点

### 主题定调

**「让 AI 编程搭子变成桌面上的卡比兽」** — 用 Claude Code 开发一个能实时感知编码状态、会聊天会提醒的桌面宠物

### 核心卖点（3 层递进）

| 层级 | 卖点 | 一句话说明 |
|------|------|-----------|
| 情感层 | 编程陪伴感 + 卡比兽人格 | 桌角有个贪吃爱睡的小家伙陪你加班、提醒你喝水 |

## 三、新功能亮点

### 卡比兽人格系统
- 52 条自言自语短语（吃东西、喝水、睡觉、陪主人、天气、鼓励、卖萌、下午茶）
- 9 种状态全部卡比兽风格标签，用颜文字 `(´・ω・`)` `(๑•̀ㅂ•́)و✧`
- 单击回复个性化，称呼用户"主人"

### 闹钟系统
- 10 个预设模板：喝水 ×4 / 吃饭 ×2 / 休息 / 下班 ×2 / 待办
- 到点弹窗提醒 + 系统提示音 + 贪睡按钮
- 自定义添加/删除/启用闹钟

### Claude 对话功能
- 双击宠物 → 搜狗输入法风格迷你浮条
- 输入问题 → 后台调用 Claude CLI → 回复以卡通气泡显示
- 气泡：奶油色圆角矩形 + 暖棕边框 + 45°猫爪装饰

### 卡通对话框
- 程序化 Canvas 绘制，不是图片资源
- 奶油色背景 + 暖棕圆角边框 + 三角尾巴 + 猫爪
- 自动适应文字长度，长文本换行

### 智能睡眠
- 10 分钟无任何交互自动睡觉
- 单击唤醒

**口播**：
> 它替我高兴一下，然后继续陪我干活。
>

**画面**：任务结束，卡比兽弹跳庆祝"开心！( ´・∀・)ﾉ"

### 第2段：灵感与架构（0:45-2:00）

**画面**：博主出镜或画外音 + 架构图动画

**口播**：
> 做这个项目的出发点很简单——独自编程时只有冷冰冰的终端输出。所以就想，能不能有个小家伙在桌面上陪你经历整个开发过程。
>
> 为什么选卡比兽？贪吃、爱睡、但又关心主人——完美的桌面宠物性格。它自称"咔比"，叫你"主人"，满口颜文字。

**画面**：动画展示完整架构

```
Claude Code Hooks → pet-notify → HTTP → Flask(:3456) → Queue → Tkinter 动画
                                                       ↑
                                               闹钟管理器 + 自言自语 + 交谈模块
```

**口播**：
> 架构简洁但功能丰富。Hooks 感知编码状态 → 状态机驱动动画是主线。三条副线并行：闹钟每 30 秒检查系统时间、自言自语每 1~3 分钟随机冒泡、交谈模块直接调 Claude CLI。

---

### 第3段：核心实现（2:00-4:30）

#### 3.1 人格系统（2:00-2:50）

**画面**：展示 STATE_CONFIG 和 CLICK_RESPONSES 代码

**口播**：
> 9 种状态，全部卡比兽风格。idle 是"呼噜噜~"，tired 是"困困...想睡觉( '・ω・)゛"，sleeping 是"呼呼呼...咔比~"。单击回复全是颜文字，比如摸鱼时是"咔比！主人你好~ 有吃的吗？(´・ω・`)"。

#### 3.2 自言自语（2:50-3:40）

**画面**：展示气泡随机弹出效果

**口播**：
> 52 条随机短语——"咔比肚子饿了... 主人有小饼干吗？"、"咔比最大的梦想就是吃饱了睡睡饱了吃~"、"主人！该喝水了！咔比也一起喝~"。每 1~3 分钟弹一条，10 秒后消失。睡觉时不弹。

#### 3.3 动画引擎（3:40-4:30）

**画面**：分屏——代码 + 实时效果

**口播**：
> 动画引擎核心是 50ms 循环，三角函数实时计算。30 多行代码，9 种动画效果——这就是程序化动画的威力。

### 第4段：功能展示（4:30-6:00）

**画面**：桌面录制，实际操作演示全流程

- **(4:30-4:50)** 双击卡比兽 → 搜狗风格浮条 → 输入"今天天气怎么样" → 卡比兽头顶弹出 Claude 回复气泡
